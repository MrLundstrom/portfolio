Continuous TD -

A semi-idle Tower Defence game for Android and IOS. It features 10 different towers with 3 upgrades each, giving (almost) endless options to build your defence. It also features 6 different game-modes, 28 permanent upgrades, leaderboads and randomly generated levels.

The game is a freemium game, which means it's free to download, but you can buy a few things with IAP or watch ads to gain rewards.

It's currently in beta testing, with a proper release planned for the end of the summer.

DotClock -

An experimental wallpaper-website, showing the current time with a pattern of dots. It features the ability to change size, color and location of the clock as well as adding background colors and switching between 12 and 24HR formats. It is available for download and usage in the Steam workshop in the application Wallpaper Engine, where it currently has over 3500 subscribers and around 90% rating.

Steam Workshop page Github repository

Transforms -

A small experiment with CSS-transforms to create a three-dimensional world without other tools. It uses css transforms to make the buildings, and it uses some javascript to change the perspective.

VG Spex -

A webapp to easily store & book food in Västgöta Nations inventory. Currently features the ability to add and delete items in inventory, book part of or whole items in inventory, and to set expiration dates that will warn once inventory goes past it's expiration date and to see booked items.
